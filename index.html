AIStatus

A real-time AI model status monitoring platform with intelligent routing SDK and local gateway proxy, helping developers and AI agents navigate the rapidly evolving AI infrastructure landscape.

  • Platform (aistatus.cc): Real-time status dashboard for 12+ AI providers, model pricing & ranking comparison, MCP server monitoring, and vector database status tracking
  • Backend: Serverless API built with Hono on Cloudflare Workers with D1 database, featuring usage leaderboard, telemetry ingestion, and zero-ops deployment
  • SDK (Python & TypeScript): Zero-config intelligent routing (route("Hello", model="claude-sonnet-4-6")), automatic provider discovery from environment variables, tier-based model fallback, and cost calculation
  • Gateway: Local transparent HTTP proxy with API key rotation, provider fallback chains, SSE streaming support, model-level health tracking, and per-request usage analytics
